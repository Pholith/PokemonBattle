
=== POKEMON BATTLE ===
Un simulateur de combat pokémons, avec sélections de team et combats contre un ordinateur.
Avec un système de combat classique et de ligue.

== BUILD AND RUN ==
Notre projet à été réalisé avec IntelliJ, pour le compiler, il faut ouvrir le projet avec IntelliJ, 
ouvrir un fichier dans l'espace de travail, et lancer la configuration MAIN 
(Il se peut quelques fois que cette erreur apparaisse 
"Error occurred during initialization of boot layer
java.lang.module.FindException: Module javafx.controls not found"
Dans ce cas, il faut réessayer plusieurs fois de lancer le projet)

== LIMITATIONS / BUGS ==

- Le mode de combat n'a pas encore été implémenté en anglais

- Une grosse limitation du projet est qu'il n'a pas au départ été pensé pour fonctionner sous format .jar
Il est donc impossible pour le moment de le compiler dans ce format, et impossible de le lancer en dehors de l'IDE.

- Un petit bug de notre projet est que certaines fenêtres ne sont pas redimensionnables.

- De nombreuses erreurs peuvent encore apparaitre lors des sauvegardes, notamment à cause de l'ajouts de nouvelles fonctionnalités.
